# Recruiter Messages — Sahid Attaf

Three versions for reaching out to recruiters or responding to AI-related
role opportunities. Target roles: AI Integration Specialist, AI Automation
Specialist, AI Workflow Builder, AI Consultant.

Do not oversell. Do not claim enterprise platform or cybersecurity experience.
Customize before sending — replace [Company] and [Role] with real values.

---

## Version 1 — Short LinkedIn Connection Note (under 300 characters)

Use for: LinkedIn connection requests. Must be short and specific.

```
Hi [Name] — I'm an AI systems builder with 5 years in workflow automation
(Python, Claude Code, GitHub). Built JobHunterGPT as a working prototype.
Open to remote AI integration roles. Happy to share my work.
```

**Character count:** ~225. Under the 300-character LinkedIn limit.

**Why it works:**
- Names the role area ("AI integration")
- Gives a specific proof ("JobHunterGPT")
- Offers to share work — low pressure, high signal
- Does not claim enterprise experience it doesn't have

## Version 2 — Medium LinkedIn DM

Use for: Following up after a connection, responding to a recruiter who has
reached out first, or cold outreach to a recruiter at a company you want to work at.

```
Hi [Name],

I saw [Company] is hiring for AI integration / workflow automation roles and
wanted to reach out directly.

I'm an AI systems builder based in Curaçao with five years of practical
experience designing and deploying AI-powered workflows — for hospitality
operations, real estate development, and business automation. The clearest
proof of that work is JobHunterGPT, a human-in-the-loop Python pipeline I
built from scratch covering keyword extraction, fit scoring, cover letter
generation, and output review. It has 70 automated tests and a live demo.

My background is non-traditional — I don't come from an enterprise environment
or hold a formal degree. What I bring is the ability to build AI systems
independently, test them rigorously, and think carefully about where automation
helps and where human review matters.

If there's a relevant opening or you work with candidates like me, I'd value
a 15-minute conversation. I can also share the GitHub repository or run the
demo live.

Thanks for your time.

Sahid Attaf
Willemstad, Curaçao | Remote preferred
LinkedIn: https://www.linkedin.com/in/sahidattaf/
```

**Why it works:**
- Mentions the company specifically (shows this isn't a mass message)
- Names specific proof (JobHunterGPT, 70 tests, live demo)
- Acknowledges the non-traditional background honestly, before they ask
- Asks for a short call, not a job — lower barrier to reply
- Offers two concrete things to share (GitHub, demo)

## Version 3 — Email-Style Recruiter Message

Use for: Direct email outreach to a hiring manager or recruiter you found via
LinkedIn, a job posting, or a referral. More formal than the DM.

**Subject:** AI Workflow Builder — Interest in [Role / Team]

```
Dear [Name],

I'm reaching out because [Company] is working on exactly the kind of AI
workflow and integration problem I've been building toward.

My background: I'm an AI systems builder based in Curaçao with five years
of practical experience translating business problems into AI-powered workflow
solutions. I work in Python, Claude Code, GitHub, and VS Code and have built
AI automation systems for hospitality operations, real estate development, and
small business clients.

The project that best demonstrates my approach is JobHunterGPT — a
human-in-the-loop Python pipeline I designed and built independently. It
covers keyword extraction from job descriptions, ATS fit scoring against a
candidate profile, cover letter generation from verified facts only, and a
structural output reviewer that runs ten checks before the package can be used.
It has 70 automated tests and never submits anything automatically. The GitHub
repository is available and I can run a live demo in five minutes.

I want to be straightforward about my background: I don't have enterprise
platform experience (Salesforce, Jira, Amazon Q) and I don't hold a formal
degree. I'm applying to roles that accept equivalent experience because I
believe the workflow design thinking and the ability to build testable,
documented AI systems is genuinely relevant — and demonstrable directly.

I completed course-based AI tool training in Claude and Gemini through Coursiv
in June 2026. I speak English, Papiamentu, Dutch, and Spanish, and I'm
available to work fully remote.

If [Role / Team] is relevant, I'd welcome a 20-minute conversation or I can
send the GitHub repository directly.

Thank you for your time.

Sahid Attaf
Willemstad, Curaçao | Remote preferred
LinkedIn: https://www.linkedin.com/in/sahidattaf/
GitHub: [add public GitHub URL]
```

**Why it works:**
- Subject line is specific to the role/company, not generic
- Opening line says why this company, not just any company
- Specific proof: module names, test count, design choices
- Gap acknowledgment in the second-to-last paragraph — honest, forward
- Certificates mentioned factually, not oversold
- Multiple language fluency mentioned — genuine differentiator for global roles
- Clear ask: conversation or GitHub link

---

## Before Sending Any Message

- [ ] Replace [Name], [Company], [Role] with real values
- [ ] Confirm the GitHub URL is public (add before sending)
- [ ] Confirm the demo is rehearsal-ready if you offer it
- [ ] Read the message out loud — does it sound like you?
- [ ] Remove any claims that are not in your resume or profile
- [ ] Do not mass-send — customize for each recipient

---

## What Not to Include in Any Recruiter Message

| Do not say | Why |
|-----------|-----|
| "I have enterprise experience" | Not verified |
| "I'm an expert in AI" | Overclaim |
| "I've worked with Salesforce / Jira / Amazon Q" | Not true |
| "I have a degree in..." | Not verified |
| "I can start immediately" | Only say this if it is true |
| "I'm the best candidate for this role" | Unprovable |
| Certificate IDs | Private — not for external sharing |
| Private phone or email | Use only in formal applications |
