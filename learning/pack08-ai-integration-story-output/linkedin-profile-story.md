## Featured Project Description — JobHunterGPT

Use this in the LinkedIn "Featured" section or as a project description.

**Title:** JobHunterGPT — Human-in-the-Loop AI Job Application Pipeline

**Description:**
```
A local Python prototype I built from scratch to automate the preparation side
of a job search while keeping a human in the loop at every step.

What it does:
- Extracts ATS keywords from job descriptions
- Scores candidate fit across keyword overlap, verified skills, location, and
  remote preference (0–100 scale)
- Generates a cover letter draft from verified resume facts only — no invented content
- Runs a structural package reviewer before any output file is used

What it does NOT do:
- Submit applications automatically
- Invent qualifications or experience
- Bypass human review at any step

8 Python modules. 70 automated tests. Runs on Python's standard library.
Human-review checkpoint at every output stage.

Available on GitHub. Demo available on request.
```
