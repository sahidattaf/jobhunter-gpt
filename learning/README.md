# JobHunterGPT 8-Pack Learning Project

A structured skill-building program created after the Proofpoint AI Integration Specialist
application review. Each pack targets a verified gap between Sahid's current profile and
the requirements of AI Integration Specialist roles.

**Honesty rule:** Every module distinguishes what can be claimed now from what must be
earned through practice. Nothing is invented. No pack graduates until the evidence exists.

## How to Use

1. Start with Pack 07 (Demo Readiness) and Pack 08 (Story) — these build on what Sahid
   already has and produce immediate results for any interview.
2. Run through Pack 01 first to settle the degree/experience question before any application.
3. Work Packs 02, 03, 04, and 06 in parallel — each can be completed in an afternoon.
4. Pack 05 builds on verified existing experience — use it to deepen and document.
5. Update `progress-tracker.csv` after every session.
6. After each pack reaches "Interview Ready," update `master_resume.txt` with verified
   new language, then regenerate the Proofpoint package.

## Rules for All Packs

- **Do not update the resume** until the Evidence column in progress-tracker.csv is filled.
- **Do not claim completion** of a pack based on reading alone. Each pack requires a
  deliverable (mini demo, document, or working artifact).
- **Do not claim Proofpoint-specific platform knowledge** (Amazon Quick, Totango).
- **Do not claim Salesforce, Jira, or enterprise data pipeline experience** until the
  relevant pack is at "Evidence Ready" or above.
